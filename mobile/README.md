# Mobile Module

This module contains the KMP + Compose Multiplatform mobile application targeting Android and iOS.

## Setup Requirements

- Android SDK (API 29+, target API 34)
- Xcode 15+ (for iOS targets)
- JDK 21

## Structure

- `composeApp/` - Shared Compose Multiplatform UI (Android + iOS)
- `shared/` - KMP shared data/domain layer
