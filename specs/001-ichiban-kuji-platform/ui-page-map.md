# PrizeDraw UI Page Map

## Part 1: 玩家端 Web（Player Web App）

### 全站 Layout

```
┌─────────────────────────────────────────────────┐
│  Top Nav Bar (sticky)                           │
│  ┌──────┬──────────────────────────┬──────────┐ │
│  │ Logo │  活動 | 市集 | 排行榜    │ 🔔 👤   │ │
│  └──────┴──────────────────────────┴──────────┘ │
├─────────────────────────────────────────────────┤
│                                                 │
│              Page Content                       │
│                                                 │
├─────────────────────────────────────────────────┤
│  Footer: 關於我們 | 條款 | 客服 | 語言切換      │
└─────────────────────────────────────────────────┘
```

- Top Nav：左 Logo + 名稱，中間主導航，右邊通知鈴鐺 + 使用者頭像下拉（我的賞品/錢包/設定/登出）
- 未登入：右邊顯示「登入」按鈕
- Mobile：漢堡選單

### 7-15. 其他管理頁面（簡述）

| 頁面 | 核心功能 |
|------|---------|
| `/trade` | 上架商品監控，可下架違規商品 |
| `/prizes` | 賞品定義列表，依活動分組，行內編輯回收價格、回收開關 |
| `/coupons` | 優惠券列表+建立表單，折扣碼生成，停用按鈕 |

| `/payments` | 金流訂單表格，狀態篩選，詳情查看 |
| `/staff` | 人員列表，建立帳號表單，角色下拉選單，啟用/停用 |
| `/audit` | 稽核紀錄表格，篩選(操作者/類型/日期)，展開看變更前後 |
| `/feature-flags` | Flag 列表 + toggle 開關，即時生效 |
| `/settings` | 交易手續費(bps輸入+%預覽)，動畫模式開關，系統資訊 |

---

## 設計原則

1. **一致性**：所有頁面使用相同的卡片、表格、badge、按鈕樣式
2. **即時反饋**：所有操作有 loading 狀態、成功/失敗 toast
3. **中文優先**：所有介面文字預設繁體中文
4. **響應式**：玩家端支援手機/平板/桌面；Admin 以桌面為主
5. **Dark mode**：玩家端支援深色模式
6. **Empty state**：每個列表頁都有空狀態提示和引導
7. **Error state**：API 失敗時顯示友善錯誤訊息+重試按鈕
